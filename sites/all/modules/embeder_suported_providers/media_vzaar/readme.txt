
-- INSTALLATION --

Go to Modules > enable Media: Vzaar.

-- CONFIGURATION --

You can find all Media settings relating to vzaar in 
Configuration > Media > File types > Video > manage file display

-- FEATURES --

* Works with multiple url patterns ie: http://vzaar.com/videos/xxxxxx, http://vzaar.com/users/UserID/videos/xxxxxx
* Also works with embed code
* Brand Text and End
